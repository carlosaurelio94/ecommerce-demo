# E-Commerce Demo — Tienda Online

Demo de tienda online con carrito de compras, filtros por categoría y pago con MercadoPago.

## Funcionalidades

- Catálogo de productos con filtros por categoría
- Modal de detalle de producto con selector de talle
- Carrito de compras deslizable
- Integración con MercadoPago
- Instalable como app (PWA)
- Diseño mobile-first, responsive

## Tecnologías

- HTML5 / CSS3 / JavaScript (vanilla)
- Vercel (deploy)
